# 10 exercícios básicos em C# para ajudar você a praticar os fundamentos da linguagem

### Exercício 1: Soma de Dois Números
- **Descrição**: Escreva um programa que peça ao usuário para inserir dois números inteiros e, em seguida, exiba a soma desses números.
- **Dica**: Use `int.Parse()` para converter a entrada de string para inteiro.

### Exercício 2: Verificação de Número Par ou Ímpar
- **Descrição**: Escreva um programa que peça ao usuário para inserir um número inteiro e verifique se o número é par ou ímpar.
- **Dica**: Use o operador `%` para verificar o resto da divisão por 2.

### Exercício 3: Calculadora Simples
- **Descrição**: Crie um programa que peça ao usuário para inserir dois números e uma operação (adição, subtração, multiplicação ou divisão) e, em seguida, exiba o resultado da operação.
- **Dica**: Use uma estrutura `switch` para selecionar a operação correta.

### Exercício 4: Tabuada
- **Descrição**: Escreva um programa que peça ao usuário para inserir um número e, em seguida, exiba a tabuada desse número de 1 a 10.
- **Dica**: Use um loop `for` para iterar de 1 a 10 e multiplicar pelo número fornecido.

### Exercício 5: Inversão de String
- **Descrição**: Crie um programa que peça ao usuário para inserir uma palavra e exiba a palavra ao contrário.
- **Dica**: Use o método `ToCharArray()` para converter a string em um array de caracteres e, em seguida, use `Array.Reverse()`.

### Exercício 6: Fatorial de um Número
- **Descrição**: Escreva um programa que peça ao usuário para inserir um número inteiro positivo e, em seguida, calcule e exiba o fatorial desse número.
- **Dica**: Use um loop `for` para multiplicar os números de 1 até o número fornecido.

### Exercício 7: Verificação de Ano Bissexto
- **Descrição**: Crie um programa que peça ao usuário para inserir um ano e verifique se é um ano bissexto.
- **Dica**: Um ano é bissexto se for divisível por 4, mas não por 100, exceto se for divisível por 400.

### Exercício 8: Soma dos Dígitos de um Número
- **Descrição**: Escreva um programa que peça ao usuário para inserir um número inteiro e calcule a soma dos dígitos desse número.
- **Dica**: Use um loop `while` para extrair os dígitos do número usando o operador `%`.

### Exercício 9: Sequência de Fibonacci
- **Descrição**: Crie um programa que exiba os primeiros 10 números da sequência de Fibonacci.
- **Dica**: A sequência de Fibonacci é definida como `F(n) = F(n-1) + F(n-2)` com `F(0) = 0` e `F(1) = 1`.

### Exercício 10: Contagem de Vogais
- **Descrição**: Escreva um programa que peça ao usuário para inserir uma frase e conte o número de vogais (a, e, i, o, u) presentes na frase.
- **Dica**: Use um loop `foreach` para iterar sobre os caracteres da string e um `if` para verificar se é uma vogal.

Esses exercícios cobrem conceitos fundamentais em C#, como manipulação de strings, loops, condicionais, e operações matemáticas básicas. Eles são uma excelente maneira de reforçar seu aprendizado e ganhar confiança na programação em C#.
